# Workflow Automation and Evidence Review

Last refreshed: **2026-05-17a**

## Current direction

Automation should reduce silent failure and manual hunting. Every scheduled or automated action should leave a visible record.

## Priority workflows

- Failed upload retry.
- Evidence review and approval/rejection notes.
- Scheduler run status and duplicate-dispatch protection.
- Report subscription delivery status.
- Payroll close delivery/confirmation.
- Signed-contract kickoff.
- Accounting close package delivery.

## Next UI improvement

Create an Error/Health Center and Media/Evidence Manager so retries, failures, evidence, and review queues are visible in one place.
