# SEO Public Page Rules

Last refreshed: **2026-05-14b**

- Use one clear H1 per exposed page.
- Use words people search for in title, main heading, alt text, and helpful link text.
- Keep local service wording accurate and visible.
- Do not promise first-page ranking; improve relevance, distance clarity, and prominence signals.
- Use structured data only where accurate and test it before deploy.
