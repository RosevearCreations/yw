# SEO Public Page Rules

Last refreshed: **2026-05-15a**

These rules apply only to public/exposed pages.

## Page basics

- One clear `<h1>` per page.
- Descriptive, concise `<title>`.
- Helpful meta description.
- Words people search for in prominent locations when accurate.
- Local/service wording for local pages.
- Useful text around images.
- Alt text for meaningful images.
- Clear crawlable links.
- No broken assets.

## Local pages

If public location/service pages are added, build them around real service areas, real services, proof content, and clear contact/business details.
